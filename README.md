# Kaguya - IA conversationnelle vocale

Kaguya est une intelligence artificielle dotée de mémoire, d'émotions et d'une voix expressive. Elle utilise :
- **faster-whisper** pour la reconnaissance vocale (STT)
- **llama-cpp-python** avec un modèle GGUF pour la génération de texte (LLM)
- **GPT-SoVITS** pour la synthèse vocale (TTS)
